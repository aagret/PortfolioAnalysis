# Portfolio Analysis

## Project Goal
Web-based portfolio analysis tool for visualizing and analyzing investment portfolio performance.

## Usage
HTML-based application for comprehensive portfolio analysis and visualization.

### Running the Application
Open the main HTML file in a web browser:
```bash
open index.html
```

### Features
- Portfolio performance analysis
- Interactive visualizations
- Risk metrics
- Return analysis

### Requirements
- Modern web browser
- Local web server (optional, for enhanced functionality)
